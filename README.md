## Projetos

### Análise de Dados de Vendas

Sistema para análise de vendas a partir de arquivos CSV.

**Funcionalidades:**

* Cálculo de faturamento total
* Identificação do produto mais vendido
* Ranking de vendas
* Cálculo de ticket médio
* Geração de gráfico de vendas
* Exportação de relatório em CSV

**Tecnologias:**

* Python
* Pandas
* Matplotlib

**Localização:**
`analise-dados/`
